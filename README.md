# Square Tetris 12x12 🟩

브라우저에서 바로 즐길 수 있는 **정사각형 블록 테트리스 게임**입니다.  
모든 블록이 **12x12 픽셀 정사각형**으로 고정되어 있으며, 점수가 올라갈수록 블록 낙하 속도가 빨라지고, 한 줄이 사라질 때 화면이 흔들리는 효과가 적용됩니다.

---

## 🎮 게임 특징

- **모든 블록 12x12 정사각형**
- 한 줄 삭제 시 **점수 +1**
- 점수 1점마다 **블록 낙하 속도 증가**
- **좌우 벽 충돌 방지** → 블록이 벽을 뚫고 나가지 않음
- 한 줄 삭제 시 **화면 흔들림(Shake) 효과**
- 게임 오버 시 **점수 초기화 및 재시작**

---

## ⬇️ 실행 방법

1. 레포지토리를 클론하거나 ZIP 다운로드
2. `index.html` 파일을 브라우저에서 열기
3. 키보드 조작으로 플레이 가능:

| 키 | 기능 |
|---|---|
| ← | 블록 왼쪽 이동 |
| → | 블록 오른쪽 이동 |
| ↓ | 블록 아래로 이동 (빠르게 낙하) |
| Space | 블록 즉시 하강 |

---

## 🛠️ 기술 스택

- HTML / CSS / JavaScript
- Canvas API 사용
- **단일 HTML 파일**로 구현 → 별도의 빌드 과정 없이 실행 가능
